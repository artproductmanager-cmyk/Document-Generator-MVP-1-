## 🎨 Новые возможности:

### Умный ValidatedInput компонент:

**Автоформатирование:**
```
Вводишь: "1234567890"
Показывает: "1234 567890"
```

**Визуальная обратная связь:**
- ✅ Зелёная галочка ✓ при правильном вводе
- ❌ Красный крестик ✗ при ошибке
- 📝 Детальные сообщения об ошибках
- 💡 Подсказки под полем

**Примеры валидации:**

#### Паспорт РФ:
```
✅ "1234 567890" → галочка
✅ "1234567890" → автоформатируется → галочка
❌ "123" → "Серия и номер паспорта должны содержать 10 цифр"
❌ "123456789" → "Серия и номер паспорта должны содержать 10 цифр"
```

#### VIN номер:
```
✅ "WVWZZZ3CZBE123456" → галочка (проверка контрольной суммы прошла)
✅ "wvwzzz3czbe123456" → автоконвертируется в верхний регистр → галочка
❌ "123" → "VIN номер должен содержать 17 символов"
❌ "WVWZZZ3CZBE12345I" → "VIN номер содержит недопустимые символы. Используйте латинские буквы (кроме I, O, Q) и цифры"
❌ "WVWZZZ3CZBE123450" → "VIN номер не прошёл проверку контрольной суммы"
```
