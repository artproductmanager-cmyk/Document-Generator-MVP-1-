```bash

```

2. **Vercel автоматически пересоберет проект** (займет 2-3 минуты)

3. **Проверьте** documentator.online через 3-5 минут

---

### ВАРИАНТ 2: Если деплоили через Netlify

1. **Залейте обновленные файлы:**
```bash
git add .
git commit -m "Fix 404 error - update vite config and netlify.toml"
git push
```

2. **Netlify автоматически пересоберет**

3. **ИЛИ** пересоберите вручную через UI:
   - Зайдите в Netlify
   - Deploys → Trigger deploy → Clear cache and deploy

---

### ВАРИАНТ 3: Если не использовали Git

**Пересоберите и переделпойте вручную:**

```bash
# 1. Убедитесь что все файлы обновлены (vite.config.ts, vercel.json)

# 2. Удалите старую папку dist
rm -rf dist

# 3. Соберите заново
npm run build

# 4. Vercel:
vercel --prod

# ИЛИ Netlify:
netlify deploy --prod --dir=dist
```
