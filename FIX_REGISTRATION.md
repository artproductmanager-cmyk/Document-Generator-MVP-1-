## 🚀 Как запустить ЛОКАЛЬНО (для тестирования):

### Шаг 1: Создай файл `.env`
```bash
# В корне проекта создай файл .env
touch .env
```

### Шаг 2: Добавь свои Supabase ключи
Открой `.env` и добавь:

```env
VITE_SUPABASE_URL=https://твой-project-id.supabase.co
VITE_SUPABASE_ANON_KEY=твой-anon-key
```

**Где взять ключи:**
1. Открой https://supabase.com/dashboard
2. Выбери свой проект
3. Settings → API
4. Скопируй:
   - **Project URL** → `VITE_SUPABASE_URL`
   - **anon public** ключ → `VITE_SUPABASE_ANON_KEY`

### Шаг 3: Запусти локально
```bash
npm run dev
```

### Шаг 4: Проверь регистрацию
1. Открой http://localhost:5173
2. Нажми "Войти" → "Регистрация"
3. Заполни форму
4. ✅ Должна сработать регистрация!
