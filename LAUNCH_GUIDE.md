## 🚀 Пошаговый гайд по запуску (Вариант 1):

### Шаг 1: Проверка Supabase (2 минуты)

1. Зайти в Supabase Dashboard
2. Проверить Edge Functions → server (должна быть зелёная)
3. Проверить Table Editor → kv_store_297fba9e (таблица есть)

### Шаг 2: Деплой Frontend (5 минут)

**Vercel (самый простой):**
```bash
# 1. Установить Vercel CLI (если нет)
npm i -g vercel

# 2. Задеплоить
vercel

# 3. Добавить production переменные в Vercel Dashboard:
VITE_SUPABASE_URL=https://xxx.supabase.co
VITE_SUPABASE_ANON_KEY=xxx

# 4. Задеплоить production
vercel --prod
```

**Netlify:**
```bash
# 1. Установить Netlify CLI
npm i -g netlify-cli

# 2. Задеплоить
netlify deploy --prod

# 3. Добавить переменные в Netlify Dashboard
```

## 🆘 FAQ:

### Q: Можно ли запускать без YooKassa?
**A:** Да! Запусти в режиме "только бесплатно", собери аудиторию, потом добавишь платежи.

### Q: Сколько времени займёт запуск?
**A:** 30 минут (если Supabase уже настроен). Деплой + тесты + домен.

### Q: Что делать с Pricing страницей без платежей?
**A:** Написать "Скоро" или собирать email'ы в waitlist.

### Q: Нужен ли Cron для сброса лимитов?
**A:** Нет, не критично. Можно добавить позже через GitHub Actions за 5 минут.

### Q: Сколько стоит хостинг?
**A:** 
- Supabase: Free tier (достаточно для старта)
- Vercel/Netlify: Free tier
- Итого: 0 руб/мес на старте!

### Q: Когда достигну 200k руб/мес?
**A:** При 5000 регистраций/мес и 10% конверсии. Реально за 6-12 месяцев с SEO + рекламой.
