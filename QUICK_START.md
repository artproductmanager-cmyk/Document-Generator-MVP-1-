## 🧪 Тестирование

### Шаг 1: Регистрация
1. Откройте `/signup`
2. Зарегистрируйте аккаунт
3. Войдите в систему

### Шаг 2: Создание документов
1. Выберите любой из 4 документов
2. Заполните форму (пошаговый мастер)
3. Скачайте Word документ
4. Документ автоматически сохранится в историю

### Шаг 3: Проверка лимитов
1. Создайте 3 документа (free tier)
2. При попытке создать 4-й — увидите сообщение о лимите
3. Перейдите на `/pricing` для покупки подписки

### Шаг 4: Личный кабинет
1. Откройте `/dashboard`
2. Увидите все созданные документы
3. Информацию о текущем тарифе
4. Счётчик оставшихся документов

## 🔧 Настройка для Production

### 1. YooKassa (5 минут)
```bash
# 1. Зарегистрироваться на yookassa.ru
# 2. Получить API ключи
# 3. Добавить в Supabase Secrets:
YOOKASSA_SHOP_ID=123456
YOOKASSA_SECRET_KEY=live_xxx
```

### 2. Cron для сброса лимитов (5 минут)
```yaml
# GitHub Actions: .github/workflows/reset-limits.yml
on:
  schedule:
    - cron: '0 0 1 * *'  # Каждое 1-е число месяца
```

### 3. Деплой (10 минут)
```bash
# Vercel (самый простой)
vercel

# Или Netlify
netlify deploy

# Или VPS
pnpm build
# Загрузить dist/ на сервер
```

Готово! Проект в production.
