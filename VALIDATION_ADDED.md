### 1. Создан файл `/src/app/utils/validation.ts`

Полный набор функций валидации для всех важных полей документов:

#### Паспортные данные РФ:
- ✅ **validatePassportSeries()** — серия паспорта (4 цифры)
- ✅ **validatePassportNumber()** — номер паспорта (6 цифр)
- ✅ **validatePassportSeriesAndNumber()** — серия + номер (10 цифр: "1234 567890")
- ✅ **formatPassport()** — автоформатирование: "1234567890" → "1234 567890"

#### VIN номер автомобиля:
- ✅ **validateVIN()** — международный стандарт:
  - 17 символов
  - Только латинские буквы (кроме I, O, Q) и цифры
  - Проверка контрольной суммы (9-й символ)
  - Детальные сообщения об ошибках
- ✅ **formatVIN()** — убирает пробелы/дефисы, приводит к верхнему регистру

#### ИНН:
- ✅ **validateINN()** — проверка ИНН:
  - 10 цифр (для организаций)
  - 12 цифр (для физ. лиц)
  - Проверка контрольной суммы
- ✅ **formatINN()** — автоформатирование

#### ОГРН:
- ✅ **validateOGRN()** — проверка ОГРН:
  - 13 цифр (для организаций)
  - 15 цифр (для ИП)
  - Проверка контрольной суммы
- ✅ **formatOGRN()** — автоформатирование

#### Телефон:
- ✅ **validatePhone()** — проверка телефона РФ
- ✅ **formatPhone()** — "+79991234567" → "+7 (999) 123-45-67"

#### Email:
- ✅ **validateEmail()** — проверка email формата

### 2. Создан компонент `/src/app/components/ui/validated-input.tsx`

Умный input с встроенной валидацией:

**Возможности:**
- ✅ Валидация при вводе (blur)
- ✅ Автоматическое форматирование (опционально)
- ✅ Иконки ✓ / ✗ для визуальной обратной связи
- ✅ Детальные сообщения об ошибках
- ✅ Красная/зелёная подсветка поля
- ✅ Текст подсказки (helpText)

**Пример использования:**
```tsx
<ValidatedInput
  label="VIN номер"
  id="carVin"
  placeholder="WVWZZZ3CZBE123456"
  value={formData.carVin}
  onChange={(e) => updateField("carVin", e.target.value)}
  validate={validateVIN}
  formatValue={formatVIN}
  helpText="17 символов, только латинские буквы и цифры"
  required
/>
```

## 🎯 Как работает:

### 1. Пользователь вводит данные:
```
Вводит: "12345678901" (в поле паспорта)
```

### 2. Автоформатирование:
```
Показывает: "1234 567890"
```

### 3. Валидация при blur (уход из поля):
```
✓ Проверяет: ровно 10 цифр?
✓ Если да → зелёная галочка ✓
✗ Если нет → красный крестик ✗ + сообщение об ошибке
```

### 4. Визуальная обратная связь:
```
✅ Правильно: зелёная рамка + галочка
❌ Ошибка: красная рамка + крестик + текст ошибки
```

## 💡 Как добавить валидацию в другие документы:

### Шаг 1: Импортируй валидацию
```tsx
import { ValidatedInput } from "../components/ui/validated-input";
import { validatePassportSeriesAndNumber, formatPassport } from "../utils/validation";
```

### Шаг 2: Замени Input на ValidatedInput
```tsx
// Было:
<Input
  id="passport"
  placeholder="серия 1234 номер 567890"
  value={formData.passport}
  onChange={(e) => updateField("passport", e.target.value)}
/>

// Стало:
<ValidatedInput
  label="Паспортные данные"
  id="passport"
  placeholder="1234 567890"
  value={formData.passport}
  onChange={(e) => updateField("passport", e.target.value)}
  validate={validatePassportSeriesAndNumber}
  formatValue={formatPassport}
  helpText="Формат: 1234 567890 (4 цифры серии + 6 цифр номера)"
  required
/>
```
